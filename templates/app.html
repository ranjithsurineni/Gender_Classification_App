Face Recognition Application


This is a web application that uses the face_recognition library to recognize faces in images. The application is built using Flask, a web framework for Python. The application allows users to upload an image and then uses the face_recognition library to recognize faces in the image. The application then displays the image with boxes around the recognized faces.

Face Recognition Pipeline

The Face Recognition Pipeline leverages Haar Cascade, Principal Component Analysis (PCA), and Support Vector Machine (SVM) to detect and classify faces in an image.

Pipeline Steps:

  1. Load Pre-trained Models:
    • Haar Cascade Classifier for face detection.
    • Pre-trained SVM model for classification.
    • PCA model for dimensionality reduction and mean face extraction.
  2. Face Recognition Process:
    1. Read Image: The image is either loaded from a file path or directly passed as an image object.
    2. Convert to Grayscale: The image is converted to grayscale to enhance face detection accuracy.
    3. Face Detection: The Haar Cascade classifier detects faces in the grayscale image.
    4. Extract & Normalize Face: The detected face is cropped, normalized (scaled between 0 and 1), and resized to 100x100 pixels.
    5. Flatten Image: The resized face is reshaped into a 1x10000 feature vector.
    6. Subtract Mean Face: The face image is adjusted by subtracting the mean face (from PCA models) to normalize variations.
    7. Apply PCA Transformation: The PCA model transforms the input face into an eigenface representation.
    8. Reconstruct Eigen Image: The transformed image is inverted to visualize the eigenface.
    9. Classification using SVM: The eigenface representation is passed to the SVM model for classification.
    10. Prediction Output: The model predicts the identity or gender along with a probability score.
    11. Draw Bounding Boxes: The detected face is highlighted with a colored bounding box and labeled with classification results.

Key Features:

  • Efficient and accurate face detection using Haar Cascade.
  • Dimensionality reduction with PCA for optimized classification.
  • Robust classification with a pre-trained SVM model.
  • Visualization of eigenfaces for interpretability.

Potential Enhancements:

  • Improve face detection in low-light conditions using Histogram Equalization.
  • Optimize face detection parameters for better accuracy.
  • Implement error handling for cases where no faces are detected.

This pipeline provides an effective solution for real-time face recognition, leveraging machine learning and deep learning techniques.
