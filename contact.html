Real-Time Language Assistant for Sign Translation (March 2024 - June 2024)

Associated with: Methodist College of Engineering and Technology

  • Objective & Motivation:
    • Sign language is one of the oldest and most natural forms of communication. However, most people do not understand sign language, and interpreters are not always available.
    • To bridge this gap, we developed a real-time sign language recognition system using neural networks that interprets hand gestures and translates them into text and speech.
    • The system is designed to recognize American Sign Language (ASL) fingerspelling using computer vision and deep learning techniques.
  • Methodology:
    • The hand gesture is first captured using a webcam and processed using image filtering techniques.
    • After preprocessing, the filtered hand gesture is passed through a deep learning classifier that predicts the corresponding sign language character or gesture.
  • Data Preprocessing & Image Processing:
    • Captured sign language images using OpenCV and preprocessed them by converting RGB to grayscale, reducing noise, and normalizing pixel values.
    • Applied image augmentation techniques (scaling, rotation, flipping) to improve model generalization.
  • Deep Learning Model Development:
    • Implemented a Convolutional Neural Network (CNN) using TensorFlow and Keras for sign recognition.
    • Trained the model to recognize ASL signs (A to Z) and common hand gestures.
    • Optimized the architecture to balance accuracy and computational efficiency for real-time performance.
  • Real-Time Gesture Recognition & Text-Speech Conversion:
    • Utilized MediaPipe for real-time hand landmark detection.
    • Processed gestures in real-time using CNN inference, translating them into text output.
    • Used Google Text-to-Speech (gTTS) API to convert recognized text into spoken words.
    • Designed an interactive Tkinter-based GUI for an improved user experience.
  • Deployment & Future Scope:
    • Optimized and deployed the model using Flask for accessibility across multiple platforms.
    • Planned future enhancements to extend support for different sign languages and improve real-time processing efficiency.

Tools & Technologies Used: Python, TensorFlow, Keras, OpenCV, MediaPipe, Google Text-to-Speech (gTTS), Flask, Tkinter, NumPy, Pandas
